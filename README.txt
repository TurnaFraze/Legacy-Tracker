
  Legacy System Character Tracker
The Legacy System Character Tracker is a web-based tool designed to assist with character development for creative writing, tabletop role-playing games (TTRPGs), and storytelling. It includes a structured questionnaire that evaluates characters across various dimensions, such as Wisdom, Connection, Action, Balance, and Identity.

 Features
- Interactive Questionnaire: Easily input scores and interpretations for different character traits.  
- Dynamic Score Calculation: Automatically calculates and displays In-Out and Left-Right scores.  
- User-Friendly Interface: Clean, modern design with hover effects and smooth transitions.  
- Offline & Online Accessibility: Runs locally or can be hosted on free platforms like GitHub Pages, Netlify, or Vercel.  
- Modular Code: Separated HTML, CSS, and JavaScript for easy maintenance and expansion.

Tech Stack
- HTML5: Structure and layout  
- CSS3: Styling and design  
- JavaScript: Dynamic score calculation and form interactions  

Usage
1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Fill in the questionnaire to generate character scores and interpretations.
